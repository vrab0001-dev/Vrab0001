## System Engine Usage

This repository includes a small helper `system_engine.py` that updates a status block in this `README.md`.

- To add the required markers to your README (safe — use `--dry-run` first):

```bash
python3 system_engine.py --init --dry-run
python3 system_engine.py --init
```

- To update the status block (use `--dry-run` to preview):

```bash
python3 system_engine.py --dry-run
python3 system_engine.py
```

Markers used by the script (place these into the README where you want the status block):

<!-- STATS_START -->
...status block managed by system_engine.py...
<!-- STATS_END -->

The script will only replace the content between `<!-- STATS_START -->` and `<!-- STATS_END -->` and will avoid touching the file if no changes are necessary. This prevents CI/watchers from retriggering updates.
